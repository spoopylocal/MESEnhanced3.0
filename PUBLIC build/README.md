# MES Enhanced 3.0

Chrome/Firefox extension that enhances the MES interface with productivity tools.

## Features

- **Quick Copy Buttons**: PN, LPN, WO, SR with one-click copying
- **MO PRM/INFO**: Fetch move order details, work orders, build names, and due dates
- **MO STATUS**: Check move order statuses (Open, Picking, Inventoried, DRP)
- **LSC Serial Checker**: Validate serial completeness (Asset #, Rack Position, Warranty)
- **IBC Serial Checker**: Inbound validation (Asset #, Rack Position, MSF fields)
- **Asset Download**: Auto-download asset sheets from attachments
- **Auto-filled Placards**: Pre-populated placard generation from work order data

## Installation

1. Navigate to `chrome://extensions` (Chrome) or `about:addons` (Firefox)
2. Enable "Developer mode"
3. Click "Load unpacked" and select the `MESEnhanced3.0` folder

## Usage

Access tools via the gear icon in the MES header or use copy buttons throughout work order pages.

Internal tool for WWT use.
